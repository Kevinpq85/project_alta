# project_alta

## Task
Aplikasi yang dikembangkan harus menyertai aspek-aspek berikut:

- Minimum 5 halaman.
- Menerapkan MVVM Architecture.
- Mengakses RESTful API.
- Menggunakan Transition
- Menggunakan Form
